# StoraChain – Final Year Project

StoraChain is a decentralized AI-powered blockchain-based disk space rental and sharing platform developed as part of the BSc (Hons) Software Engineering degree - final Year project.

## Project Objective

The system aims to create a peer-to-peer storage marketplace where users can:

- Rent unused disk space
- Securely store files using encryption
- Utilize blockchain for transparent storage agreements
- Apply AI for intelligent matchmaking and optimization

## Current Development Stage

Phase 1 – Foundation Setup:
- Project structure initialized
- Backend environment setup in progress
- Database configuration planned
- Authentication module development started

## Technology Stack

Frontend: React.js  
Backend: Node.js + Express  
Database: MongoDB Atles 
Blockchain: Ethereum (Solidity Smart Contracts)  
AI Module: Python (Planned)
